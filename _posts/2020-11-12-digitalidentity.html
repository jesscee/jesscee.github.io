An exciting project to be a part of. Building the official public website for the Australian Government's Digital Identity was very rewarding.

Website design and development

The Digital Identity website was developed using Drupal 8. The design and theme was customised using the Australian Government Design System as guidance, ensuring that WCAG 2.1 Guidelines (Level AA) were adhered to. Making sure content is accessibile should always be a top priority for any website project.

The website content, information architecture, and the overall website structure was tested with users to see if users could successfully (and easily) find the information they were seeking. With the user research findings, we were able to make evidence-based content changes.
